**1. Modify `templates/macros/methodology.html`** — Replace the entire macro with this updated version that adds a `sparql-actions` div after the `<pre><code>` block (per D-04, D-05, D-06, D-07):

```html
{% macro methodology_note(notes, plot_key) %}
{% if plot_key in notes %}
{% set note = notes[plot_key] %}
<details class="methodology-note">
    <summary>Methodology</summary>
    <div class="methodology-content">
        <p><strong>What this measures:</strong> {{ note.description }}</p>
        <p><strong>Data source:</strong> {{ note.data_source }}</p>
        <p><strong>Known limitations:</strong> {{ note.limitations }}</p>
        {% if note.sparql %}
        <details class="sparql-query">
            <summary>View SPARQL Query</summary>
            <pre><code>{{ note.sparql }}</code></pre>
            <div class="sparql-actions">
                {% if sparql_endpoint is defined and sparql_endpoint %}
                <a href="{{ sparql_endpoint }}?query={{ note.sparql | urlencode }}"
                   target="_blank" rel="noopener"
                   class="sparql-run-btn"
                   title="Opens the Virtuoso SPARQL editor with this query pre-filled">
                    Run on Endpoint
                </a>
                {% endif %}
                <button class="sparql-copy-btn" onclick="
                    navigator.clipboard.writeText(this.closest('.sparql-query').querySelector('code').textContent);
                    var btn=this; btn.textContent='Copied!';
                    setTimeout(function(){btn.textContent='Copy Query';}, 2000);
                ">Copy Query</button>
            </div>
        </details>
        {% endif %}
    </div>
</details>
{% endif %}
{% endmacro %}
```

- `target="_blank" rel="noopener"` on the link (per D-07)
- `title` attribute with tooltip text from UI-SPEC copywriting contract
