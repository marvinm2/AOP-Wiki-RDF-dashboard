  <action>
Read every `def plot_latest_*` function in plots/latest_plots.py. For each function, evaluate against the 8-dimension binary rubric (per D-03):

**Rubric dimensions (evaluate from source code per D-06):**
1. **Color correctness** (HEAVY): Does the plot use appropriate coloring?
   - Single-series data should use single #307BBF (marker_color='#307BBF')
   - Multi-series categorical data should use BRAND_COLORS['palette'] or color_discrete_sequence
   - OECD-related plots should use BRAND_COLORS['oecd_status'] mapping
   - Property type plots should use BRAND_COLORS['type_colors'] mapping
   - Check: What Plotly constructor is used? (px.bar with color= is always categorical; go.Bar with marker_color= is explicit)

2. **Chart type appropriateness** (HEAVY): Is this the right chart type?
   - Categorical comparison -> bar chart
   - Distribution -> box/histogram/violin
   - Single metric -> indicator or simple bar
   - Composition -> stacked bar

3. **Data table usefulness** (LIGHT): Does _plot_data_cache store meaningful data?
   - Check if function writes to _plot_data_cache and what DataFrame it stores

4. **Title quality** (LIGHT): Is fig.update_layout(title=...) descriptive and consistent?

5. **Axis labels & units** (LIGHT): Are xaxis_title and yaxis_title set with appropriate labels?

6. **Legend clarity** (LIGHT): Is showlegend set appropriately? Are trace names meaningful?

7. **Layout & spacing** (LIGHT): Are height, margin, barmode settings reasonable?

8. **Tooltip quality** (LIGHT): Does hovertemplate or hoverinfo provide useful formatted data?

**Classification (per D-04):**
- **FIX NOW**: Dimension 1 (color correctness) OR dimension 2 (chart type) fails
- **FIX LATER**: Only dimensions 3-8 fail
- **SKIP**: All 8 dimensions pass

**Color decision (per D-05):**
For each plot, state one of:
- "single #307BBF" -- plot shows one series/category, should use uniform blue
- "categorical palette" -- plot shows multiple distinct categories, palette is appropriate
Include a one-line rationale explaining why.

**Wiring check:**
Cross-reference each function name against template data-plot-name attributes in latest.html and index.html. Mark as "wired" or "unwired".

**Colorblind cross-reference:**
For plots using categorical colors, note if any confusable pairs from COLORBLIND-FINDINGS.md appear in the plot's color assignments.
