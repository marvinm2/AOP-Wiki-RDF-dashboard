   '#29235C', '#E6007E', '#307BBF', '#009FE3', '#EB5B25', '#93D5F6', '#9A1C57', '#45A6B2', '#B81178', '#005A6C', '#64358C'

2. Also include the OECD status colors (8 colors including #999999) and type_colors (6 colors) for completeness.

3. Implement sRGB linearization using the standard piecewise function (per D-10, Vienot/Brettel model):
   - if c <= 0.04045: c / 12.92
   - else: ((c + 0.055) / 1.055) ** 2.4

4. Implement inverse sRGB (linear to sRGB):
   - if c <= 0.0031308: 12.92 * c
   - else: 1.055 * c ** (1/2.4) - 0.055

5. Implement Vienot 1999 deuteranopia simulation using the matrix:
   [[0.29275, 0.70725, 0.00000],
    [0.29275, 0.70725, 0.00000],
    [-0.02234, 0.02234, 1.00000]]
   Pipeline: hex -> sRGB [0,1] -> linearize -> matrix multiply -> clip [0,1] -> inverse sRGB -> clip [0,1] -> hex

6. Implement RGB to CIELAB conversion via XYZ (D65 illuminant):
   - sRGB to XYZ matrix:
     [[0.4124564, 0.3575761, 0.1804375],
      [0.2126729, 0.7151522, 0.0721750],
      [0.0193339, 0.1191920, 0.9503041]]
   - D65 reference white: [0.95047, 1.00000, 1.08883]
   - CIE f function: if t > 0.008856: t^(1/3), else: 7.787*t + 16/116

7. Implement CIEDE2000 delta E formula (Sharma, Wu, Dalal 2005). This is ~60-80 lines of numpy math. Key steps: compute C', h', T, delta_L', delta_C', delta_H', S_L, S_C, S_H, R_T rotation term.

8. For all 11 palette colors:
   a. Simulate deuteranopia appearance (produce simulated hex)
   b. Convert both original and simulated to CIELAB
   c. Print table: color name, original hex, simulated hex, Lab values

9. For all 55 pairwise comparisons of the 11 simulated palette colors:
   a. Compute CIEDE2000 delta E between each pair's simulated Lab values
   b. Flag pairs where delta E < 10 as "confusable under deuteranopia"
   c. Print sorted table: color A, color B, delta E, confusable (yes/no)

10. Print a summary section listing only confusable pairs (delta E < 10).

The script should print all output to stdout in a markdown-friendly format so it can be captured into COLORBLIND-FINDINGS.md.
