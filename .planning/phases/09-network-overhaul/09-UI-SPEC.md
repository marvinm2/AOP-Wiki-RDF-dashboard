## Interaction Contract

### Graph Rendering

| Interaction | Behavior |
|-------------|----------|
| Page load | Graph renders instantly using pre-computed positions (`layout: {name: 'preset'}`). No layout animation delay. |
| Node positions | Deterministic across visits (server-side `spring_layout(seed=42)`). Same data = same layout. |
| Node color | Colored by biological role: MIE=#EB5B25, KE=#307BBF, AO=#E6007E. |
| Node sizing | Unchanged -- sized by selected centrality metric (PageRank/Degree/Betweenness/Closeness). |

### Role Filter Dropdown

| Interaction | Behavior |
|-------------|----------|
| Default state | "All Roles" selected, all nodes visible. |
| Select MIE/KE/AO | Only nodes of selected role visible. Edges with hidden endpoints are hidden. |
| Reset Filters button | Returns dropdown to "All Roles", shows all nodes and edges. |

### Type Legend

| Interaction | Behavior |
|-------------|----------|
| Display | Always visible in filter panel bar, right-aligned after metric selector. |
| Interactivity | None -- purely informational. Not clickable. |

### Info Panel (Node Click)

| Interaction | Behavior |
|-------------|----------|
| Type badge | Shows full role name with role-colored background badge (MIE=orange, KE=blue, AO=magenta). |
| Community row | Remains visible -- shows community number as informational data point. |
| All other rows | Unchanged (degree, betweenness, closeness, pagerank, neighbors, wiki link). |

## Layout Specification

### Filter Panel (modified)

```
+-------------------------------------------------------------------+
| Role: [All Roles v]  Size by: [PageRank v]  [Reset]  * MIE * KE * AO |
+-------------------------------------------------------------------+
```

- Role dropdown replaces Community dropdown at the same position (first filter group).
- Type legend sits right-aligned via `margin-left: auto` on the `.type-legend` container.
- Legend dots are 10x10px circles with 4px gap to label text, 8px gap between legend items.
- Reset Filters button position unchanged.

### Info Panel Type Badge (modified)

```
+---------------------------+
| Type: [Molecular Initiating Event]  <-- orange badge for MIE
| Type: [Key Event]                   <-- blue badge for KE
| Type: [Adverse Outcome]             <-- magenta badge for AO
+---------------------------+
```

Badge styling inherits existing `.type-badge` base (3px 10px padding, 12px border-radius, 12px font-size, 600 weight, 0.5px letter-spacing). New classes `.mie` and `.ao` add role-specific background colors.
