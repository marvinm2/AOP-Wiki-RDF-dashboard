**2. Create `templates/base.html`:**
This is the master template all pages inherit from. Structure:

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>{% block title %}AOP-Wiki RDF Dashboard{% endblock %}</title>
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <link rel="stylesheet" href="{{ url_for('static', filename='css/main.css') }}">
    <link rel="stylesheet" href="{{ url_for('static', filename='css/lazy-loading.css') }}">
    {% block head_extra %}{% endblock %}
</head>
<body>
    <nav class="main-nav">
        <div class="nav-container">
            <a href="/" class="nav-brand">AOP-Wiki RDF Dashboard</a>
            <div class="nav-links">
                <a href="/snapshot" class="nav-link {% if active_page == 'snapshot' %}active{% endif %}">Database Snapshot</a>
                <a href="/trends" class="nav-link {% if active_page == 'trends' %}active{% endif %}">Historical Trends</a>
                <a href="/network" class="nav-link {% if active_page == 'network' %}active{% endif %}">Network Analysis</a>
                <a href="/about" class="nav-link {% if active_page == 'about' %}active{% endif %}">About</a>
            </div>
            <div class="nav-version-selector">
                {% block version_selector %}{% endblock %}
            </div>
        </div>
    </nav>

    <main class="page-content">
        {% block page_header %}{% endblock %}
        {% block content %}{% endblock %}
    </main>

    <footer class="main-footer">
        <div class="footer-container">
            <div class="footer-branding">
                <img src="{{ url_for('static', filename='images/vhp4safety-logo.png') }}"
                     alt="VHP4Safety" class="footer-logo"
                     onerror="this.style.display='none'">
                <span class="footer-project-name">AOP-Wiki RDF Dashboard</span>
            </div>
            <div class="footer-links">
                <a href="https://aopwiki.cloud.vhp4safety.nl" target="_blank" rel="noopener">SPARQL Endpoint</a>
                <a href="https://github.com/marvinm2/AOP-Wiki-RDF-dashboard" target="_blank" rel="noopener">GitHub</a>
                <a href="https://github.com/marvinm2/AOP-Wiki-RDF-dashboard/issues" target="_blank" rel="noopener">Report an Issue</a>
            </div>
            <div class="footer-funding">
                Developed within the <a href="https://vhp4safety.nl" target="_blank" rel="noopener">VHP4Safety</a> project, funded by NWO
            </div>
        </div>
    </footer>

    <script src="{{ url_for('static', filename='js/lazy-loading.js') }}"></script>
    {% block scripts_extra %}{% endblock %}
</body>
</html>
```

Key design decisions:
- Navigation is STATIC (scrolls with page), not fixed — data-dense pages need screen real estate
- Version selector block is empty by default — only latest.html fills it
- Footer has three rows: branding, links, funding acknowledgment
- `active_page` variable set by each Flask route via `render_template(..., active_page='snapshot')`
- `onerror` on logo img hides it gracefully if file is missing
- Plotly CDN script is NOT in base.html — only pages that need it include it via `head_extra` block

**3. Add navigation and footer CSS to `static/css/main.css`:**

Footer styles:
- Background: `var(--color-footer-bg)` (#29235C deep purple)
- Text: `var(--color-footer-text)` (white)
- Links: white with underline on hover
- Logo: max-height 40px
- Layout: centered, stacked rows with padding

Also add `.page-content` with `max-width: var(--content-max-width)` and centered margin for consistent content width.

Basic tablet responsive: Add `@media (max-width: 900px)` rules to stack nav links, reduce padding, allow content to fill width.
