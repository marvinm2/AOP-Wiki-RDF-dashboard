What is AOP-Wiki?

The AOP-Wiki is the primary repository for Adverse Outcome Pathways (AOPs), which describe mechanistic links between molecular initiating events and adverse outcomes relevant to risk assessment. AOPs consist of Key Events (KEs) connected by Key Event Relationships (KERs), forming directed networks that capture biological knowledge.

This dashboard monitors the RDF (Resource Description Framework) version of AOP-Wiki data, enabling SPARQL-based analytics across quarterly snapshot releases. The RDF data captures AOPs, Key Events, Key Event Relationships, Stressors, biological processes, cellular components, and ontology annotations.
